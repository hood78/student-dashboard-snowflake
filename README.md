# 🎓 Student Performance Dashboard using Snowflake

## 📌 Overview
An interactive analytics dashboard built using **Snowflake + Streamlit + SQL** to analyze student performance data.

This project helps visualize:
- Top performing students
- Weak students
- Average marks by branch
- Students by city
- KPI metrics

---

## 🚀 Features

✅ Interactive Dashboard  
✅ Branch / City / Gender Filters  
✅ Top Students Leaderboard  
✅ Weak Students Detection  
✅ Charts & Insights  
✅ Connected to Snowflake Database

---

## 🛠 Tech Stack

- Snowflake
- SQL
- Streamlit
- Python
- Pandas

---

## 📂 Project Structure

student-dashboard-snowflake/
│── app.py
│── students.sql
│── student.csv
│── README.md
│── screenshots/

---

## ⚙️ Setup

1. Create database in Snowflake  
2. Run `students.sql`  
3. Create Streamlit App  
4. Paste `app.py` code  
5. Run dashboard
